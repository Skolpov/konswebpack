8.3. Актуальный список бандлеров

webpack

Мощный и гибкий бандлер, используемый для сборки веб-приложений.

Документация webpack.

Vite

Быстрый сборщик с поддержкой ES-модулей. Подходит для разработки с минимальным запуском бандлера.

Документация Vite.

Parcel

Легковесный и быстрый бандлер с простым API и автоматическим управлением зависимостями.

Документация Parcel

Gulp

Инструмент для автоматизации различных задач, в том числе сборки проектов.

Документация Gulp.

Rollup

Минималистичный бандлер. Подходит для сборки библиотек и модулей.

Документация Rollup.

esbuild

Быстрый бандлер с фокусировкой на производительность (скорость) сборки.

Документация esbuild.
