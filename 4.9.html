Шаг 1. Начало работы

Перед нами проект в нерабочем состоянии. Мы не можем запустить сервер и собрать проект, так как получаем при выполнении команд сообщения об ошибках и отсутствии зависимостей.

Давайте по порядку.

При вводе команды npm run build мы получаем одну ошибку.

При вводе npm start также получаем ошибку, и проект не запускается.

Как видно, обработчик, навешенный на расширения .js и .jsx, не справляется и не может распарсить код, преобразовав его в чистый JS.
